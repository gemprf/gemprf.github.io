GEM-pRF Software & Methodology

To cite the GEM-pRF software or methodology, please cite our research paper published in Medical Image Analysis:

Mittal et al. (2025), GEM-pRF: GPU-Empowered Mapping of Population Receptive Fields for Large-Scale fMRI Analysis

Documentation Website

If you used the GEM-pRF documentation website (e.g., the XML configuration generator), you can cite it using the following DOI:

Mittal et al. (2025), GEM-pRF Software Documentation for Population Receptive Field (pRF) Mapping

Software Version

When specifying the software version used in your research, please refer to the version number listed on the GEM-pRF PyPI package manager page. This ensures reproducibility and clarity about which version of the software was used.
