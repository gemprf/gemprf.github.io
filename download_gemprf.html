Download GEM-pRF

GEM-pRF can be downloaded in two ways, depending on your access and workflow.

1. Download from PyPI (recommended)

The easiest and most common way to install GEM-pRF is directly from PyPI: 

pip install gemprf

The latest release is available here: PyPI – gemprf.

After installation, see Running for instructions on running GEM-pRF.

2. Source Code

This option is useful for contributors or advanced users who want to inspect the implementation, debug, or modify the code. For most users, installing from PyPI is the recommended and simplest method.

3. Sample Configuration File

Once you have GEM-pRF installed, you'll need a configuration file to run your analysis. A sample configuration file is provided below to give you a sense of how GEM-pRF configurations work.

Don't worry if it looks intimidating at first! XML configuration files are actually quite straightforward once you understand the basic structure. Most settings are well-commented, and you only need to modify a few key parameters for your analysis.

If you have any questions or run into difficulties, don't hesitate to reach out to the GEM-pRF authors. We're here to help!
