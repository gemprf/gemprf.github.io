Download GEM-pRF

GEM-pRF can be downloaded in two ways, depending on your access and workflow.

1. Download from PyPI (recommended)

The easiest and most common way to install GEM-pRF is directly from PyPI: 

pip install gemprf

The latest release is available here: PyPI â€“ gemprf.

After installation, you can run GEM-pRF exactly as described in the Using GEM-pRF example.

2. Download from GitHub (restricted access)

Users who have been granted access to the private GEM-pRF GitHub repository may also download the source code directly from there.

This option is mainly for contributors or advanced users who want to inspect the implementation, debug, or modify the code.

For most users, installing from PyPI is the recommended and simplest method.
