GEM-pRF

Using GEM-pRF

Once gemprf is installed and you have a matching CuPy build installed, running GEM-pRF is very simple. The latest release of gemprf is available on PyPI.

Quick example

Import the package and call the run function with your XML configuration file:

import gemprf as gp
gp.run("path/to/your_config.xml")

A sample configuration file is available here: sample_config.xml .

This works in scripts, Jupyter notebooks, or any Python IDE.
