GEM-pRF

Example — Run GEM-pRF (basic)

This example shows a minimal workflow to run GEM-pRF using the packaged API.

Files used

  • sample_configs/sample_config.xml — example XML config.
  • Input data (place your preprocessed fMRI files in the paths referenced by the XML).

Steps

  1. Install GEM-pRF and a matching CuPy build: 
    pip install gemprf
pip install cupy-cuda12x  # replace 12x with your CUDA version
  2. Run GEM-pRF from Python: 
    import gemprf as gp
gp.run("sample_configs/sample_config.xml")
  3. Check the output folder specified in the XML for results and logs.

Troubleshooting (quick)

  • If GEM-pRF errors on import, verify CuPy and CUDA versions match (nvidia-smi / nvcc --version).
  • For CUDA/CuPy compatibility see: CuPy install.
