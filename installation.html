GEM-pRF

Installation

GEM-pRF requires GPU access for data processing. It uses CUDA libraries to access and process data on NVIDIA GPUs.

⚠️ Please ensure your system has a compatible NVIDIA GPU available.

Step 1. Set up GPU environment

Before installing dependencies, configure your GPU environment:

  • Install the CUDA SDK if not already installed.
  • Check your installed CUDA version: 
    nvcc --version
nvidia-smi
  • Install a CuPy version compatible with your CUDA version: 
    pip install cupy-cuda12x  # Replace 12x with your CUDA version

Step 2. Install Python dependencies

# Create or activate your preferred environment
pip install -r requirements.txt

Step 3. Download GEM-pRF code

git clone https://github.com/siddmittal/GEMpRF.git
cd GEMpRF

✅ Once installed, proceed to Running GEM-pRF.
