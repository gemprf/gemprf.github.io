Step-by-Step Guide

Option A: Run from Terminal

  • Open a terminal (e.g. Anaconda Prompt) and activate your environment.
  • Navigate to the GEMpRF folder and execute: 
    python run_gem.py PATH_TO_YOUR_XML_CONFIG_FILE

Option B: Run from IDE (e.g. VS Code)

  • Open the downloaded GEMpRF folder in VS Code.
  • Edit run_gem.py to specify your XML config file path.
  • Run the script directly from the IDE.
